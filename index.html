Лабораторная работа №1

Язык разметки гипертекста HTML

Цель работы

Изучить основы языка разметки гипертекста HTML 5.

Задание

Создать html-документ, разметить его в соответствии с вариантом (см. далее по тексту). Для ВСЕХ ВАРИАНТОВ в разметке html-документа использовать:

  1. тег <!DOCTYPE>
  2. тег <meta> для определения кодировки текста;
  3. тег <!-- --> комментария;
  4. новые семантические теги стандарта HTML5 для структурирования документа;
  5. теги:
    1. <p> - абзац,
    2. <br> - пере-
      нос на другую строку,
    3. <div> - выделение блока,
    4. <span> - стрйоные элементы,
    5. <hr> - черта для отделения,
    6. один из <h1> - <h6> -

      заголовок,

    7. <pre> -   неформатированный                       текст
      ,
    8. <mark> - выделение жёлтым цветом,
    9. <time> - контейнер для времени и даты,
    10. <details> - сворачиваемый виджет
      <summary> - видимый заголовок для тега <details>;

      Тег <summary> и <details> используются вместе

  6. <span>продемонстрировать</span> <span>различие</span>
    <div>блочных</div>
    <div>и строчных</div>
    тегов на примере тегов <div> и <span>;
  7. теги <figure>, <img>, <figcaption> для вставки изображения;
  8. тег <a> для разметки гиперссылок, разметить ссылки:
    1. на другой документ;
    2. в пределах размечаемого документа.
  9. теги для разметки списка, таблицы, формы в соответствии с вариантом (см. далее по тексту).
